Objetivo

Este projeto consiste na construção e consolidação de uma base de dados relacional voltada para a análise das instituições financeiras brasileiras. O banco reúne dados do IFDATA (Banco Central) e da CVM para viabilizar consultas, visualizações e análises estratégicas de forma organizada e eficiente.

A base foi estruturada para ser utilizada como backend de outros projetos analíticos, incluindo o Dashboard Power BI do Projeto 01.

Escopo do Projeto
O que foi desenvolvido?
  • Modelagem relacional: criação de tabelas normalizadas para dados cadastrais, balanços, DREs e indicadores das instituições financeiras.
  • Extração (Extract): coleta automatizada das bases do IFDATA (BCB) e das DFPs padronizadas da CVM.
  • Limpeza e Tratamento (Transform): padronização de nomenclaturas, tratamento de valores nulos, remoção de duplicatas e consistência de séries históricas.
  • Carga (Load): inserção dos dados tratados nas tabelas do SQL Server com controle de versão e atualização incremental.
  • Views e Procedures: construção de visualizações prontas para consumo no Power BI e análises diretas via SQL.
